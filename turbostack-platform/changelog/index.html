Welcome to the TurboStack Platform changelog. This page provides a transparent record of all updates, improvements, and fixes made to the platform. You will find details on new features, performance enhancements, security updates, and resolved issues.

v0.6.11

Released: 10-10-2025

  1. DB Credentials Extra database users can now be created, 2 roles can be defined read only and admin
  2. Backup restore Daily backups can now be restored on both linux and Windows servers. It can be either to a specified location or directly in place.
