**MedusaJS** is an open-source, headless eCommerce platform built in JavaScript and Node.js, designed to give developers and merchants a highly flexible, API-first foundation for building custom online commerce experiences. It features a modular architecture with composable commerce modules for products, carts, orders, inventory, discounts, multi-region/currency support and more, plus a customizable admin dashboard and APIs that let you build your storefront with any frontend framework.
